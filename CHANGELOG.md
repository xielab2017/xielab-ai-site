# Changelog

## v0.3.0-editorial-console — 2026-07-13

- Rebuilt `/admin/` as a section-by-section Editorial Console.
- Added structured forms for homepage, contact, openings, members, research, publications, patents, news and tools.
- Added add, edit, delete, member-to-alumni, filtering, browser drafts and JSON preview controls.
- Added deterministic `[CMS]` issue payloads and an authorized GitHub Actions write layer.
- Added optional OpenRouter bilingual polishing restricted to approved narrative fields.
- Added collection-specific matching, normalization, sorting, validation and audit reports.
- Preserved review-first publishing: every console update creates a pull request before deployment.
- Added the Editorial Console operations and security guide.

## v0.2.0-ai-cms — 2026-07-13

- Added AI Content Studio at `/admin/`.
- Added `.docx` member intake with paragraph, table and image extraction.
- Added OpenRouter bilingual profile generation and deterministic member matching.
- Added review pull requests for member additions, revisions and alumni transitions.
- Added AI-assisted GitHub issue intake for news, research, tools, publications and patents.
- Added PubMed, optional Google Scholar, patent and translation workflows.
- Added full structured-content validation and member merge unit tests.
- Added GitHub Pages deployment and `www.xielab.net` CNAME configuration.
